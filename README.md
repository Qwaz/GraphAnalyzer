GraphAnalyzer
=============
#frame.txt
- 시작년도 끝년도

- 노드 관련 데이터 개수(L)

- 간선 관련 데이터 개수(K)

지원하는 형식은 int, Number, String입니다.

각각의 약자인 i, n, s를 가장 앞에 적고 그 뒤에 속성 이름을 적습니다.
예를 들어 문자열 형태의 '이름' 속성을 정의하려면 's이름'형태로 정의합니다.

#graph.txt
- 노드개수(N)

- 초기 데이터 입력이 들어온다.(N * L)

- 변화의 개수(A)

발생년도 / 노드 번호 / 데이터 입력(L개) / 변화가 없다면 - 입력

#edge.txt
- 간선 개수(M)

- 초기 데이터 입력이 들어온다.(M * K)

- 변화의 개수(B)

발생년도 / 첫번째 노드, 두 번째 노드 / 데이터 입력(M개) / 변화가 없다면 - 입력
